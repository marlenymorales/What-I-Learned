CSS Checkup

In this lesson I learned about:

Classes-

A class is an attribute which specifies one or more class class names for an HTML element.

ID-

An ID defines an identifier which must be unique in the whole document.

Margin or Padding

Margin is the space around an element border, while paddiing is the space between an element's content.
